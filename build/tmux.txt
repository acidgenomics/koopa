# Installation

## conda method

Install using [conda][] package manager. Recommended if you don't have sudo access.

```bash
conda install tmux
```

## yum method

Install using [yum] package manager. Recommended when working on a VM with sudo access.

```bash
sudo yum install tmux
```

Now that [tmux][] is installed, here's how to check the version.

```bash
tmux -V
```

## Build and install from source

On some Linux distros, tmux can be pretty out of date. Here's how to build from source on a system that supports [yum][].

```bash
sudo yum install yum-utils
sudo yum-builddep tmux
```

```
version="2.8"
wget "https://github.com/tmux/tmux/releases/download/${version}/tmux-${version}.tar.gz"
tar -xzvf "tmux-${version}.tar.gz"
cd "tmux-${version}"
```

```bash
./configure --prefix="/usr/local"
make
sudo make install
```

Alternatively, here's how to install without `sudo`.

```bash
./configure --prefix="${HOME}/.local"
make
make install
```

### Without root access

This guide is modified from [ryin's gist](https://gist.github.com/ryin/3106801).

```bash
set -e
```

Create the directories.

```bash
mkdir -p "${HOME}/.local" "${HOME}/tmux_tmp"
cd "${HOME}/tmux_tmp"
```

#### Install libevent

```bash
LIBEVENT_VERSION"2.0.19"
wget "https://github.com/downloads/libevent/libevent/libevent-${LIBEVENT_VERSION}-stable.tar.gz"
tar -xvzf "libevent-${LIBEVENT_VERSION}-stable.tar.gz"
cd "libevent-${LIBEVENT_VERSION}-stable"
./configure --prefix="${HOME}/.local" --disable-shared
make -j
make install
cd ..
```

#### Install ncurses

```bash
NCURSES_VERSION="5.9"
wget "ftp://ftp.gnu.org/gnu/ncurses/${NCURSES_VERSION}-5.9.tar.gz"
tar -xvzf "ncurses-${NCURSES_VERSION}.tar.gz"
cd "ncurses-${NCURSES_VERSION}"
./configure --prefix="${HOME}/.local"
make -j
make install
cd ..
```

#### Install tmux

```bash
TMUX_VERSION="2.8"
wget -O "tmux-${TMUX_VERSION}.tar.gz" "https://github.com/tmux/tmux/archive/${TMUX_VERSION}.tar.gz"
tar -xvzf "tmux-${TMUX_VERSION}.tar.gz"
cd "tmux-${TMUX_VERSION}"
./autogen.sh
./configure CFLAGS="-I${HOME}/.local/include -I${HOME}/.local/include/ncurses" LDFLAGS="-L${HOME}/.local/lib -L${HOME}/.local/include/ncurses -L${HOME}/.local/include"
CPPFLAGS="-I${HOME}/.local/include -I${HOME}/.local/include/ncurses" LDFLAGS="-static -L${HOME}/.local/include -L${HOME}/.local/include/ncurses -L${HOME}/.local/lib" make
cp tmux "${HOME}/.local/bin"
cd ..
```

Clean up files.

```bash
cd ~
rm -rf "${HOME}/tmux_tmp"
```
