Flags:
    
    -h, --help
        Help mode.
    -V, --version
        Version string.
    
Commands:
    
    check
        System checks.
    info
        System information.
    list
        List of programs accessible in '$PATH'.
    update
        Update koopa installation to latest version.
    variable
        Get internal koopa variable.
        For example, get recommended 'vim' version.
